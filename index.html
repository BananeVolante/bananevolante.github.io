Professional experiences

A 6 months graphics programmer internship in EFICAD

I developed the base of a CAD application's new graphic engine, using OpenGL 4.6 and C++20. I was tasked to implement various features, for example displaying text or transparent materials. For each of these features, I listed the different methods available and chose one, taking into account the needs of the company, the development and maintenance complexity, the performance, the visual result...
This engine offered a massive performance boost compared to the previous one, which was written in OpenGL 1.X. During 4 months, I worked with another intern whom I trained and supervised.

A 4 months internship in a laboratory in Ikoma, Japan

I participated in the 2023 IPIN competition. The objective was to track the position of a phone carried by a person walking in an indoor environment. To perform this task, I had access to the phone's Inertial Measurement Unit (Accelerometer + Gyroscope + Magnetometer) and to around 100 Bluetooth beacons. Instead of relying on a state of the art solution, the objective was to find new methods.
My tasks mainly consisted in analyzing the available data to find points that could be improved, formulating possible improvements, discussing them with my tutor, implementing those improvements and evaluating their effectiveness.
I finished third out of eight participants.

A compiler for an esoteric language

As a school project, in a group of 4, we developed a compiler for a programming language that exclusively uses binary trees. We parsed the source code using ANTLR, converted it into an Abstract Syntax Tree then into three-address code and finally into C++, which we compiled. We also implemented some code optimization techniques.
